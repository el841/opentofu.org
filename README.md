# OpenTofu Website

## Stack

- Docusaurus
- React
- TypeScript
- Tailwind CSS

## Getting Started

1. Clone the repository

```bash
git clone git@github.com:opentofu/opentofu.org.git
```

2. Install dependencies

```bash
npm i
```

3. Fetch the documentation

```bash
make
```

4. Start the development server

```bash
npm run start
```
